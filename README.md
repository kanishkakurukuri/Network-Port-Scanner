# Simple Network Port Scanner
## About
This is a simple port scanner built using Python and Tkinter.  
The project was created to understand basic networking concepts like ports and how systems communicate over a network.
It provides a small GUI where you can enter a target IP address and scan a range of ports.
## Features
- Scan ports of any IP address or hostname  
- Simple graphical interface  
- Shows open ports  
- Displays common services (like HTTP, SSH)  
- Option to stop scanning  
## Technologies Used
- Python  
- Tkinter (for GUI)  
- Socket module  
- Threading
